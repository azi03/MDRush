## **Introduction**
>MDRush是一个在线的markdown语法翻译工具。
可以通过如下地址：
[https://mdrush.herokuapp.com](https://mdrush.herokuapp.com/)

#### **实现原理**

vuejs进行数据绑定+marked插件进行markdown语法解析。


#### **使用**

1、在左侧编辑符合markdown语法的文本可以再右边被实时的渲染出来。
